# Chandrayaan-3: The Complete Journey

An interactive cinematic p5.js simulation of India’s historic Chandrayaan-3 mission — from launch at Sriharikota to lunar landing and Pragyan rover deployment.

---

#  Features

##  Mission Phases

### Phase 1 — Mission Briefing
- ISRO-style intro screen
- Animated glowing UI
- Mission timeline
- Interactive launch button

### Phase 2 — Rocket Launch
- Countdown sequence
- LVM3 rocket animation
- Smoke particle effects
- Dynamic launch physics
- Crowd & launch infrastructure

### Phase 3 — Separation & Orbit
- Booster separation
- Fairing separation
- Propulsion module separation
- Earth parking orbit
- Orbital manoeuvre visualization
- Satellite animation

### Phase 4 — Moon Mission
- Trans-lunar injection
- Lunar orbit insertion
- Vikram lander descent
- Pragyan rover deployment
- Rover movement using keyboard controls

---

#  Controls

| Action | Control |
|---|---|
| Start Mission | Click Launch Button |
| Ignite Rocket | Mouse Click |
| Stage Separation | Mouse Click |
| Begin Lunar Descent | Mouse Click |
| Move Pragyan Rover | ← → Arrow Keys |
